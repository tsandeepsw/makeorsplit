# MakeOrSplit 🎨

MakeOrSplit is a React web application that lets you **split images into multiple parts** by drawing divider lines, and also **create collages** by combining images together.  

---

## ✨ Features
- ✂️ **Split Mode**: Upload an image, draw lines, or use presets and download all split parts as a ZIP file.
- 🖼️ **Collage Mode**: Arrange multiple images into a custom collage.
- ⚡ **Fast & Free**: Runs entirely in your browser, no server required.
- 📦 **Download Ready**: Automatically packages split images into a ZIP file.
